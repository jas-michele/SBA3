# Clipboard Landing Page

This project recreates the Clipboard landing page using HTML, SCSS, and Bootstrap. 

# Technologies Used

- HTML
- SCSS
- Bootstrap
- Responsive Design
- Git & GitHub

# Features

- Responsive layout that adjusts for desktop and mobile screens
- Styled buttons for iOS and Mac
- Organized footer with nav links and social icons
- SCSS styling compilied into CSS 

# What I learned 

This project helped reinforce how responsive layputs work using Bootstrap's grid system and breakpoints. I practiced using SCSS for styling and compiling it into CSS while building a layout that adapts across screen sizes.

Using the design reference and style guide helped me break each section of the page down before starting to code. I was able to plan what elements, spacing, and styling I needed to implement before writing the HTML and SCSS.

# Future Improvements 

- Refine spacining to better match the design 
- Add hover for interactive elements

# Reflection 

The challenge that held me up the most during this project was accidentally editing the compilied CSS file instead of the SCSS file. My changes were not compiling correctly and I was unable to see uppdates in the browser. This caused me to delete and restart the watch command multiple times before realizing the issue.

If given more time, I would focus on improving my spacing and alignment so the layout more closely matches the design example. Also spend more time refining responsiveness.
